# opsdroid skill dance

A skill for [opsdroid](https://github.com/opsdroid/opsdroid) to add dancing GIFs at appropriate times.

## Requirements

None.

## Configuration

None.

## Usage

#### `dance`

Inserts a dancing GIF.

> user: I love to dance!
>
> opsdroid: http://i.imgur.com/lnvJm9a.gif
